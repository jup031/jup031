Terms and Definitions

HTML table
HTML tables allow one to arrange data into cells within rows and columns, defined by the HTML tags <table> </table>. Within the table tags, you can use HTML tags <caption></caption> to create a title for the table. HTML tags <th></th> are used to assign specific cells within your table as headers. You can also assign an entire row as the header row by using HTML tags <thead></thead>. The "thead" tags should be used in conjunction with HTML tags <tbody></tbody> and <tfoot></tfoot> to group body content as well as the footer content. HTML tags <tr></tr> create rows for your table. Within every "tr" tag, there should be at least one <td></td> tag, which is for cells that will have table data.
Colspan vs. Rowspan
The colspan attribute defines the number of columns that a specific cell should span. Rowspan attribute defines the number of rows a cell should span.
Cell spacing vs. Padding
Cell spacing is a CSS property that defines the space between each cell. You can change this by using "border-spacing" property in your css file. Padding is a css property that defines the space between the cell edges and the cell content. It can be changed by using the "padding" property within your css file.
How to define table borders using CSS
The "border" property is used to change the outline of, for example, your HTML table. You may change the color and sizing of the border.
How to control border widths using CSS
The "border-width" CSS property establishes the width of an element's four borders. You can have 1-4 values for this property to manipulate each side of the border. You must declare the "border-style" property before the "border-width" property.
